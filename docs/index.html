No local setup needed — runs entirely on GitHub
  1. Get your own copy — click "Use this template" (or fork) on the repo to create a private copy under your account.
  2. Enable Actions — in your new repo go to Settings → Actions → General and set Workflow permissions to Read and write.
  3. Run the workflow — go to Actions → "Draw Commit Art" → Run workflow, fill in the same values you configured above, and click Run workflow. Use your GitHub verified email so commits appear on your graph.
  4. Wait ~10 min — GitHub processes the new commits and your contribution graph updates automatically.
